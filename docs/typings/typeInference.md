# 类型推断

TypeScript 能根据一些简单的规则推断（检查）变量的类型，因为这些规则足够的简单，你可以训练你的大脑去发现这些安全/不安全的代码（它很快发生在我和我的队友身上）。

::: tip
流动的类型，是我在大脑中想象的类型信息流。
:::

## 定义变量

变量的类型，由定义推断：

```ts
let foo = 123     // foo 是 'number'
let bar = 'hello' // bar 是 'string'

foo = bar         // Error: 不能将 'string' 分配给 `number`
```

这是一个关于从右向左流动类型的示例。
